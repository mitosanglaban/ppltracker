Google Sheets Sync

Requires internet connection. Paste your Google Apps Script Web App URL below. Sync is additive and non-destructive — only new or updated records are written.

Apps Script Setup Instructions

1. Open Google Sheets and go to Extensions → Apps Script.

2. Paste the doPost(e) function that receives JSON: {date, records:[{name,status,timeIn,timeOut,...}]}

3. Deploy as a Web App with access set to "Anyone".

4. Copy the deployment URL and paste it above.

The system sends one tab per month (e.g. "2026-06") and a final "Attendance Summary" tab.
